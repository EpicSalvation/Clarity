# User-Requested Features

Features in this list will only be implemented if specifically requested by users. They are not part of the current development roadmap.

## Audio & Video

- [ ] **Audio Playback Support**
  - [ ] Background music for slides
  - [ ] Audio cues/sound effects
  - [ ] Volume control
  - [ ] Fade in/out options
  - [ ] Continue across slides option

- [ ] **Video Playback Controls**
  - [ ] Start time / end time trimming
  - [ ] Playback speed control
  - [ ] Audio track enable/disable per slide
  - [ ] Loop point configuration

- [ ] **Lower-Third Animations**

  - [ ] Fade in/out
  - [ ] Typewriter effect
  - [ ] Configurable timing
